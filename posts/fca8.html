    整数二分

    1. 步骤

    二分并不是一种单调性的体现，而是一种规则的划分。使得左边的都符合，或者右边的都符合

    以数组寻找一个数为例

    函数头

    public static int seekLeft(int[] q,int goal);
public static int seekRight(int[] q,int goal);

    **q**：数组

    **goal**：所查找的值（可能不存在）

    1：左右边界的确立

    int l=0,r=q.length-1;

    一般而言左右边界时依据所查找的具体范围来确定的

    2：确定判断函数

    public static boolean check(int mid){
    /*
    * 根据条件进行实际判断
    */
}

    3：循环查找

    //模板一
while(l<r){
    int mid=l+r>>1;
    if(check(mid)){
        r=mid;
    }else {
		l=mid+1;
    }
}

//模板二
while(l<r){
    int mid=l+r+1>>1;
    if(check(mid)){
        l=mid;
    }else {
        r=mid-1;
    }
}

    循环跳出条件为l<r，即跳出时l==r，方便后续的讨论

    模板一：用于求第一个**>=goal**的情况

    模板二：用于求第二个**<=goal**的情况

    4：讨论后续

    //此时l==r，继续根据题目进行讨论

    可以判断此时q[l]==goal，如果不等则说明goal不存在数组中；其他题目依据实际情况而定

    2. 模板

    判断函数

    public static boolean check(int mid){
    /*
    * 根据条件进行实际判断
    */
}

    模板一

    public static int seekLeft(int[] q,int goal){
    int len=q.length;
    int mid=0,l=0,r=len-1;

    while(l<r){
        mid=l+r>>1;
        if(check(mid)){
            r=mid;
        }else {
            l=mid+1;
        }
    }
    return r;
}

    模板二

    public static int seekRight(int[] q,int goal){
    int len=q.length;
    int mid=0,l=0,r=len-1;

    while(l<r){
        mid=l+r+1>>1;
        if(check(mid)){
            l=mid;
        }else {
            r=mid-1;
        }
    }
    return l;
}

    记忆口诀

    r=mid，则mid=l+r>>1

    r=mid-1，则mid=l+r+1>>1，r少一则mid来补一
