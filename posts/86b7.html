    Linux学习笔记

    实用指令

    1. cd /目录名 切换目录
    2. mkdir 文件名 创建文件
    3. rmdir 文件名 删除文件夹
    4. 上下键 历史命令
    5. cal 显示日历 date 显示时间

    专题介绍

    1.vim 和 vi 编辑器

    1. vim "文件名.后缀"
      • 若已经存在，则打开；否则创建新的文件
      • 此时进入一般模式
    2. i 或 a
      • 在进入一般模式之后，按下则可以进入插入模式
    3. ESC + :
      • 先按住ESC 再按住:
      • q退出，需要先进行保存
      • q!强制退出，即不保存退出
      • wq保存之后退出
    4. 粘贴当前行p
    5. 拷贝当前行yy，拷贝当前行向下5行 5yy
    6. 删除当前行dd，删除当前行向下5行 5dd
    7. 查找某个单词 /+关键词，按n查找下一个
    8. 设置行号set nu，取消行号set nonu
    9. 定位到文档最末行G，最首行gg
    10. 按u取消上一步操作
    11. 使光标挑战到某一行（首先需要有行号）数字+shift + g

    2.关机&重启命令

    • sync 把内存数据同步在磁盘上，建议关机前使用

    • shutdown

      shutdown -h now 立即关机

      shutdown -h 1 1分钟之后关机

      shutdown -r now 立即重启

      shutdown -c 取消命令

    • halt 等价于关机

    • rebot 重启系统

    3.用户管理

    • 用户注销

      logout 在图形界面无效，只能在远程登陆注销账户

    • 切换用户

      • su - 用户名 注意横线之间的空格，切换到root用户直接su
      • sudo 可以申请在常规用户下使用更高权限
      • exit 返回切换之前的用户

    • 添加用户 useradd [选项] 用户名

      • useradd -g 用户组 用户名

    • 设置密码 passwwd 用户名

    • 删除用户 userdel 用户名

      • 删除但保存家目录 user 用户名
      • 删除同时删除目录 userdel -r 用户名

    • 查询用户信息 id 用户名

    • 创建用户组 groupadd 组名

    • 删除组 groupdel 组名

    • 修改用户组 usermod -g 用户组 用户名

    • 用户和组的配置文件

      • /etc/passwd 用户的配置文件
      • /etc/shadow 口令的配置文件，密码和登录信息加密处理
      • /etc/group 组的配置文件

    4.运行级别&找回用户密码

    • 运行级别

      0：关机

      1：单用户【找回丢失的密码】

      2：多用户状态没有网络服务

      3：多用户状态有网络服务

      4：系统保留【未启用】

      5：图形界面

      6：重启

    • 配置文件/etc/inittab/

    • 找回密码

      • 思路：进入到单用户模式->修改密码

      • 进入单用户模式不需要密码就可以进入root模式

      • 步骤

        • CentOS 7 版本
        • 开机进入选项菜单，立即点击任意键暂停
        • 按e，编辑选项菜单
        • 移动上下键至linux16核心命令行
        • 在核心命令行最后加入rd.break
        • 再按下Ctrl + x，进入单用户模式s
        • 再输入以下命令 switch_root:/# mount –o remount,rw /sysroot
          switch_root:/# chroot /sysroot
        • 设定密码passwd 用户名
        • 再输入命令sh-4.2# touch /.autorelabel
        • 退出exit ，退出两次
        • 重启完成
        • 参考链接
        • CentOS 6 版本
        • 开机进入选项菜单，按e编辑选项菜单
        • 选中kernel开头的项，按e
        • 进入新界面之后输入 s (空格+s)，按Enter
          • 上述方法还可以直接输入1，再按Enter
        • 再次回到之前界面，按b
        • 进入新界面
        • 输入passwd 用户名即可
        • 重启即可

    • 切换运行级别

      • 直接切换

        • init 数字

      • 设置默认级别

        • CentOS 6 版本

          • 直接vim /etc/inittab
          • 修改最底部的数字即可
          • 重启

        • CentOS 7 版本

          • systemctl get-default 查看默认运行级别方式

          • systemctl set-default 指令 设置默认运行指令

          • init级别 systemctl target
            0 shutdown.target
            1 emergency.target
            2 rescure.target
            3 multi-user.target
            4 无
            5 graphical.target
            6 reboot.target
          • systemctl 命令 说明
            systemctl get-default 获得当前的运行级别
            systemctl set-default multi-user.target 设置默认的运行级别为mulit-user
            systemctl isolate multi-user.target 在不重启的情况下，切换到运行级别mulit-user下
            systemctl isolate graphical.target 在不重启的情况下，切换到图形界面下

          • 参考链接

    5.帮助指令

    • man指令

      • 基本语法：man (命令或配置文件)例如 man ls
      • 向后翻一屏：space 向后翻一行：Enter
      • 向前翻一屏： b 向前翻一行：k
      • 退出：q

    • help指令

      • 基本语法：help 命令 例如help cd
      • 获得shell内置命令的帮助信息
      • 拓展：ls --help

    6.文件目录类

    • pwd 显示当前工作的绝对路径

    • ls指令

      • 显示目录下的文件

      • 基本语法：ls [选项] [目录和文件]

      • 常用选项

        • -a 显示当前目录的所有文件和目录，包括隐藏的
        • -l 以列表的方式显示信息

    • cd指令

      • 切换目录

      • 基本语法：cd [参数]

      • 常用参数

        • 绝对路径和相对路径
        • cd ~ 或者 cd ，直接回到自己家目录
        • cd .. ，回到当前目录的上一级目录

    • mkdir 指令

      • 创建目录
      • 基本语法：mkdir 目录名
      • 常用选项
        • -p 可以创建多级目录

    • rmdir 指令

      • 删除目录
      • 基本语法：rmdir [选项] 要删除的空目录
      • 常用选项
        • 删除非空目录 rm -rf 目录名（-r 和 -f 的合写）

    • touch 指令

      • 创建空文件
      • 基本语法：touch 文件名称

    • cp 指令

      • 拷贝文件到指定目录
      • 基本语法：cp [选项] source dest
      • 常用选项
        • -r 递归复制整个文件夹
        • 当发现路径下有相同名字文件，会询问
          • \cp -r source dest 强制覆盖，不于询问

    • rm指令

      • 移除文件或目录
      • 基本语法：rm [选项] 要删除的文件或目录
      • 常用选项
        • -r 递归删除整个文件夹
        • -f 强制删除不提示

    • mv 指令

      • 移动文件与目录或者重命名
      • 基本语法
        • mv oldNameFile newNameFile 重命名（即在本文件夹下操作）
        • mv /temp/movefile /targetFolder 移动文件（移动到不同文件夹）

    • cat 指令

      • 查看文件内容（只读）
      • 基本语法
        • cat [选项] 要查看的文件
      • 常用选项
        • -n 显示行号
        • | more 分页显示
          • Enter 下一行 Space 下一页

    • more 指令

      • 以全屏幕的方式按页显示文本文件的内容

      • 基本语法

        • more 要查看的文件

      • 快捷键

        • Space 向下翻一页
        • Enter 向下翻一行
        • q 立即退出
        • Ctrl + F 向下滚动一屏
        • Ctrl + B 返回上一屏
        • = 输出当前行号
        • :f 输出文件名和当前行号

    • less 指令

      • 分屏查看文件内容，与more 指令类似。less 显示文件内容时，并不是一次将整个文件加载之后才显示 ，而是根据显示需要加载内容，对于显示大型文件有较高的效率
      • 基本语法
        • less 要查看的文件
      • 快捷键
        • Space || pagedown 向下翻动一页
        • pageup 向上翻动一页
        • /字串 向下查找 n 向下查找 N 向上查找
        • ?字串 向下查找 n 向下查找 N 向上查找
        • q 退出应用

    • > 指令 和 >> 指令

      • > 输出重定向 和 >> 追加
      • 基本语法
        • ls -l >文件 （列表的内容写入文件中，覆盖写）
        • ls -al >>文件 （列表的内容追加到文件的末尾）
        • cat 文件1>文件2 （将文件1的内容覆盖到文件2）
        • echo "内容" >>文件
