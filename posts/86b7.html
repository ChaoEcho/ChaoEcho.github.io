Linux学习笔记

实用指令

  1. cd /目录名 切换目录
  2. mkdir 文件名 创建文件
  3. rmdir 文件名 删除文件夹

专题介绍

1.vim 和 vi 编辑器

  1. vim "文件名.后缀"
    • 若已经存在，则打开；否则创建新的文件
    • 此时进入一般模式
  2. i 或 a
    • 在进入一般模式之后，按下则可以进入插入模式
  3. ESC + :
    • 先按住ESC 再按住:
    • q退出，需要先进行保存
    • q!强制退出，即不保存退出
    • wq保存之后退出
  4. 粘贴当前行p
  5. 拷贝当前行yy，拷贝当前行向下5行 5yy
  6. 删除当前行dd，删除当前行向下5行 5dd
  7. 查找某个单词 /+关键词，按n查找下一个
  8. 设置行号set nu，取消行号set nonu
  9. 定位到文档最末行G，最首行gg
  10. 按u取消上一步操作
  11. 使光标挑战到某一行（首先需要有行号）数字+shift + g

2.关机&重启命令

  • sync 把内存数据同步在磁盘上，建议关机前使用

  • shutdown

    shutdown -h now 立即关机

    shutdown -h 1 1分钟之后关机

    shutdown -r now 立即重启

    shutdown -c 取消命令

  • halt 等价于关机

  • rebot 重启系统

3.用户管理

  • 用户注销

    logout 在图形界面无效，只能在远程登陆注销账户

  • 切换用户

    • su - 用户名 注意横线之间的空格，切换到root用户直接su
    • sudo 可以申请在常规用户下使用更高权限
    • exit 返回切换之前的用户

  • 添加用户 useradd [选项] 用户名

    • useradd -g 用户组 用户名

  • 设置密码 passwwd 用户名

  • 删除用户 userdel 用户名

    • 删除但保存家目录 user 用户名
    • 删除同时删除目录 userdel -r 用户名

  • 查询用户信息 id 用户名

  • 创建用户组 groupadd 组名

  • 删除组 groupdel 组名

  • 修改用户组 usermod -g 用户组 用户名

  • 用户和组的配置文件

    • /etc/passwd 用户的配置文件
    • /etc/shadow 口令的配置文件，密码和登录信息加密处理
    • /etc/group 组的配置文件

4.运行级别&找回用户密码

  • 运行级别

    0：关机

    1：单用户【找回丢失的密码】

    2：多用户状态没有网络服务

    3：多用户状态有网络服务

    4：系统保留【未启用】

    5：图形界面

    6：重启

  • 配置文件/etc/inittab/

  • 找回密码

    • 思路：进入到单用户模式->修改密码

    • 进入单用户模式不需要密码就可以进入root模式

    • 步骤

      • CentOS 7 版本

      • 开机进入选项菜单，立即点击任意键暂停

      • 按e，编辑选项菜单

      • 移动上下键至linux16核心命令行

      • 在核心命令行最后加入rd.break

      • 再按下Ctrl + x，进入单用户模式s

      • 再输入以下命令 switch_root:/# mount –o remount,rw /sysroot
        switch_root:/# chroot /sysroot

      • 设定密码passwd 用户名

      • 再输入命令sh-4.2# touch /.autorelabel

      • 退出exit ，退出两次

      • 重启完成

      • 参考链接

      • CentOS 6 版本
      • 开机进入选项菜单，按e编辑选项菜单
      • 选中kernel开头的项，按e
      • 进入新界面之后输入 s (空格+s)，按Enter
        • 上述方法还可以直接输入1，再按Enter
      • 再次回到之前界面，按b
      • 进入新界面
      • 输入passwd 用户名即可
      • 重启即可

  • 切换运行级别

    • 直接切换

      • init 数字

    • 设置默认级别

      • CentOS 6 版本

        • 直接vim /etc/inittab
        • 修改最底部的数字即可
        • 重启

      • CentOS 7 版本

        • systemctl get-default 查看默认运行级别方式

        • systemctl set-default 指令 设置默认运行指令

        • init级别 systemctl target
          0 shutdown.target
          1 emergency.target
          2 rescure.target
          3 multi-user.target
          4 无
          5 graphical.target
          6 reboot.target
        • systemctl 命令 说明
          systemctl get-default 获得当前的运行级别
          systemctl set-default multi-user.target 设置默认的运行级别为mulit-user
          systemctl isolate multi-user.target 在不重启的情况下，切换到运行级别mulit-user下
          systemctl isolate graphical.target 在不重启的情况下，切换到图形界面下

        • 参考链接
