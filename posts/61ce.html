    浮点数二分

    一. 思路

    浮点数二分，常见用于求开根号，开立方根

    函数头

    public static double floatBinary(double num)

    num：传入的浮点数

    0：先讨论特殊情况—负数

    if(num<0){
	return floatBinary(-num);
}

    避免重复讨论，负数先当作正数处理，输出时在具体判断

    1：确定左右边界

    double l=0,r=Math.max(num,1);

    l**：因为已经排除负数*的干扰，所以左边界从0*开始即可

    *r**：当num<1时，开出平方后的答案要>num，所以当num<1时将右边界取为1*，其余情况为自身即可

    2：确定判断函数

    3：循环判断

    方法一—设置r-l的区间范围

    double eps=1e-8;
while(r-l>eps){
    double mid=(l+r)/2;
    if(check(mid)){
        r=mid;
    }else {
        l=mid;
    }
}

    通常设置的区别范围标识要比所输出的数量级多两位（最常使用）

    方法二—-设置固定循环次数

    boolean check(double mid){/*检查mid是否满足一定性质*/}

double eps=1e-8;
for(int i=0;i<100;i++){
    double mid=(l+r)/2;
    if(check(mid)){
        r=mid;
    }else {
        l=mid;
    }
}

    固定循环一定次数，一般情况都可以搞定

    模板

    public static double seekAns(double num){
    if(num<0){
        return seekAns(-num);
    }

    double l=0,r=Math.max(num,1);
    double goal=1e-8;

    while(r-l>goal){
        double mid=(l+r)/2;
        if(check(mid)){
            r=mid;
        }else {
            l=mid;
        }
    }
    return l;
}
